Description

This project is a simple Amazon-style e-commerce website built using HTML and CSS. It focuses on basic UI structure, product layout, navigation bar, and responsive design fundamentals. The main goal of this project was to understand how real e-commerce websites are structured.

Challenges

Creating a clean product grid layout and aligning sections properly using only HTML and CSS was challenging. Making the navigation bar and product cards visually balanced also required a lot of adjustments.

Future Improvements

This project can be improved by adding JavaScript features like cart system, filtering, and search. Later it can be upgraded using React and backend integration for full e-commerce functionality.
