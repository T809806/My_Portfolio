# 🌐 Personal Portfolio Website

This is my personal portfolio website built using **HTML, Tailwind CSS, and JavaScript**.  
It is a responsive single-page application (SPA) showcasing my skills, projects, education, and contact information.

---

## 🚀 Features

- Responsive design (Mobile, Tablet, Desktop)
- Hamburger menu for mobile navigation
- Smooth section navigation (SPA style)
- Downloadable resume button
- Projects showcase section
- Education & skills display
- Contact form with social links

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)
- Font Awesome Icons

---

## 📌 How to Run

1. Download or clone the repository
2. Open `index.html`
3. Or use **Live Server (VS Code)** for better experience
